# Getting Started

This guide covers logging in, navigating the platform, and understanding your C3PAO dashboard.

---

## Logging In

Navigate to the C3PAO portal and sign in with your organization credentials.

## Platform Navigation

The main navigation sidebar provides access to all platform sections.

{/* <!-- SCREENSHOT: c3pao-nav-sidebar — Full sidebar navigation showing all menu items: Dashboard, Engagements, Team, Workload, Profile --> */}
![Navigation Sidebar](/img/screenshots/c3pao/nav-sidebar.png)

| Menu Item | Description |
|-----------|-------------|
| **Dashboard** | Home page with key metrics and recent activity |
| **Engagements** | All customer assessment requests and active engagements |
| **Team** | Team member management and credentials |
| **Workload** | Team workload distribution and assignment overview |
| **Profile** | Organization settings, marketplace listing, and credentials |

## Dashboard Overview

The dashboard is your home base — it shows a snapshot of your organization's assessment activity.

{/* <!-- SCREENSHOT: c3pao-dashboard-full — Full dashboard view showing all four metric cards, alert banner, and recent engagements list --> */}
![C3PAO Dashboard](/img/screenshots/c3pao/dashboard.png)

### Key Metrics

The top of the dashboard displays four metric cards:

| Metric | Description |
|--------|-------------|
| **New Requests** | Assessment requests awaiting your response |
| **In Progress** | Active assessments currently underway |
| **Total Active** | All engagements not yet completed |
| **Completed** | Finished assessments |

{/* <!-- SCREENSHOT: c3pao-dashboard-metrics — Close-up of the four metric cards at the top of the dashboard --> */}
![Dashboard Metrics](/img/screenshots/c3pao/dashboard-metrics.png)

### New Request Alert

When you have pending requests, a prominent alert banner appears with a direct link to review them.

{/* <!-- SCREENSHOT: c3pao-new-request-alert — The yellow/orange alert banner showing "You have X new assessment requests" with action button --> */}
![New Request Alert](/img/screenshots/c3pao/new-request-alert.png)

### Recent Engagements

Below the metrics, you'll see your most recent engagements with status badges and quick-access links.

{/* <!-- SCREENSHOT: c3pao-recent-engagements — The recent engagements list on the dashboard showing org names, package names, status badges, and CMMC levels --> */}
![Recent Engagements](/img/screenshots/c3pao/recent-engagements.png)
