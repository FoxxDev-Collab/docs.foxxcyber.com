# Marketplace Profile

Optimizing your presence in the C3PAO Marketplace.

## Marketplace Overview

The C3PAO Marketplace helps organizations find and request assessments from qualified C3PAOs. Your marketplace profile determines how you appear to potential customers.

![Screenshot: Marketplace listing](/img/screenshots/c3pao/marketplace-listing.png)

## Profile Components

### Display Information

What customers see when browsing:

| Element | Description |
|---------|-------------|
| Organization Name | Your C3PAO name |
| Logo | Your organization logo |
| Tagline | Brief description (100 characters) |
| Description | Detailed overview (500 characters) |
| Rating | Customer satisfaction score |
| Verification Badge | CMMC-AB verified status |

### Capabilities

Showcase your assessment capabilities:

- **CMMC Levels** - Levels you're authorized to assess
- **Industry Experience** - Sectors you've worked with
- **Team Size** - Number of qualified assessors
- **Languages** - Languages supported

### Service Areas

Indicate where you provide services:

- Geographic regions covered
- Remote assessment capabilities
- On-site assessment availability

## Editing Your Profile

Navigate to **Settings > Marketplace Profile**:

1. Click **Edit Profile**
2. Update information
3. Preview changes
4. Publish updates

![Screenshot: Edit marketplace profile](/img/screenshots/c3pao/edit-marketplace.png)

## Optimizing Your Listing

### Compelling Description

Write a description that:
- Highlights your expertise
- Mentions differentiators
- Uses professional language
- Addresses customer needs

### Showcase Experience

Highlight relevant experience:
- Number of assessments completed
- Industries served
- Average time to certification

### Response Metrics

Customers see your responsiveness:
- Average response time
- Request acceptance rate
- Customer satisfaction scores

## Managing Availability

### Capacity Settings

Indicate your current availability:

| Setting | Options |
|---------|---------|
| Accepting Requests | Yes / Limited / No |
| Estimated Lead Time | Weeks until availability |
| Assessment Slots | Number available |

### Calendar Integration

Optionally share availability:
- Block unavailable dates
- Show open assessment windows
- Integrate with booking requests

## Customer Reviews

### Review Display

Customer reviews appear on your profile:
- Star rating
- Written feedback
- Assessment date

### Responding to Reviews

Address customer feedback:
1. Click on the review
2. Add your response
3. Maintain professional tone
4. Submit response

## Analytics

View marketplace performance:

- **Profile Views** - How often your profile is viewed
- **Request Rate** - Percentage of views resulting in requests
- **Conversion Rate** - Requests that become engagements

Navigate to **Analytics > Marketplace** for detailed metrics.

![Screenshot: Marketplace analytics](/img/screenshots/c3pao/marketplace-analytics.png)

## Visibility Settings

Control your marketplace presence:

| Setting | Effect |
|---------|--------|
| **Public** | Visible to all customers |
| **By Invitation** | Only visible with direct link |
| **Hidden** | Not listed in marketplace |

## Best Practices

- **Complete profile** - Full profiles rank higher in searches
- **Quality images** - Professional logo and imagery
- **Update regularly** - Keep availability and capabilities current
- **Respond quickly** - Fast response times improve visibility
- **Encourage reviews** - Satisfied customers can leave reviews
