# Managing Engagements

Learn how to manage your C3PAO assessment engagements in Bedrock CMMC.

## Engagements Overview

The Engagements section displays all your assessment requests and active engagements.

Navigate to **Engagements** from the main menu to view:

- Pending assessment requests
- Active engagements
- Completed assessments
- Cancelled engagements

![Screenshot: Engagements list](/img/screenshots/c3pao/engagements-list.png)

## Engagement Statuses

| Status | Description |
|--------|-------------|
| **Requested** | New request awaiting your response |
| **Pending** | Under review by your organization |
| **Accepted** | Engagement confirmed, assessment not yet started |
| **In Progress** | Active assessment underway |
| **Completed** | Assessment finished and results submitted |
| **Cancelled** | Engagement was cancelled by either party |

## Viewing Engagement Details

Click on any engagement to view:

- **Organization Information** - Company details, contacts, scope
- **Package Details** - ATO package being assessed, CMMC level
- **Timeline** - Key dates and milestones
- **Assessment Team** - Assigned assessors
- **Progress** - Current status of control reviews

![Screenshot: Engagement detail view](/img/screenshots/c3pao/engagement-detail.png)

## Filtering and Sorting

Use the filters to find specific engagements:

- **Status** - Filter by engagement status
- **CMMC Level** - Filter by assessment level
- **Date Range** - Filter by request or scheduled dates
- **Assessor** - Filter by assigned lead assessor
