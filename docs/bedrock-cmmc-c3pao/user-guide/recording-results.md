# Recording Results

Documenting and submitting your assessment findings.

## Assessment Completion

Before recording final results, ensure:

- All controls have been reviewed
- All evidence has been evaluated
- All findings are documented
- Team consensus is reached
- POA&Ms have been assessed

## Results Overview

Navigate to **Assessment Results** to view the summary:

![Screenshot: Results overview](/img/screenshots/c3pao/results-overview.png)

## Assessment Determinations

### Passed

Recommend certification when:
- All controls are Met or N/A (with justification)
- No critical findings exist
- Security posture supports CMMC level

### Conditional

Recommend conditional certification when:
- Minor findings exist but are documented in POA&Ms
- Remediation plans are reasonable
- Core security controls are implemented
- Risk is acceptable with POA&Ms

### Failed

Do not recommend certification when:
- Critical findings exist
- Multiple controls are Not Met
- Evidence is significantly lacking
- Security posture is inadequate

## Recording Findings

### Finding Structure

For each Not Met control, document:

| Field | Description |
|-------|-------------|
| Control | CMMC practice identifier |
| Finding | Description of the deficiency |
| Severity | Critical, High, Medium, Low |
| Evidence | What was reviewed |
| Impact | Business/security impact |
| Recommendation | Suggested remediation |

### Severity Guidelines

| Severity | Criteria |
|----------|----------|
| **Critical** | Fundamental security gap, immediate risk |
| **High** | Significant deficiency, substantial risk |
| **Medium** | Notable gap, moderate risk |
| **Low** | Minor deficiency, limited risk |

## Submitting Results

### Review Phase

1. Click **Review Results**
2. Verify all findings are complete
3. Confirm overall recommendation
4. Add summary observations

### Approval Phase

1. Lead assessor reviews all determinations
2. Team validates findings
3. Quality review (if required by your C3PAO)
4. Final approval

### Submission

1. Click **Submit Results**
2. Confirm submission
3. Results are shared with the organization
4. Certification process proceeds

![Screenshot: Submit results](/img/screenshots/c3pao/submit-results.png)

## Post-Submission

After submission:

- Organization receives results notification
- Results are locked from further editing
- Certification recommendation is processed
- Assessment record is preserved

## Results Report

Generate the formal assessment report:

1. Navigate to **Reports**
2. Select **Assessment Report**
3. Choose format (PDF recommended)
4. Download or share directly

The report includes:
- Assessment scope and methodology
- Control-by-control results
- Finding details
- Overall recommendation
- Assessor information

## Amendments

If results need correction after submission:

1. Contact your C3PAO administrator
2. Document the required change
3. Follow your organization's amendment process
4. Updated results are submitted

## Best Practices

- **Be thorough** - Ensure all findings are complete before submission
- **Be clear** - Write findings that are actionable
- **Be fair** - Base determinations on evidence
- **Be timely** - Submit results within expected timeframes
