# Assessor Credentials

Managing assessor certifications and qualifications.

## Credential Management

Track and maintain assessor credentials within your C3PAO organization.

Navigate to **Settings > Credentials** to manage qualifications.

![Screenshot: Credentials management](/img/screenshots/c3pao/credentials-list.png)

## Credential Types

### CMMC Credentials

| Credential | Description |
|------------|-------------|
| CCA | Certified CMMC Assessor |
| CCP | Certified CMMC Professional |
| Lead Assessor | Lead Assessment Authority |

### Supporting Certifications

Related certifications that support assessment activities:
- CISSP, CISM, CISA
- Security+ and other CompTIA
- ISO 27001 Lead Auditor
- Industry-specific certifications

## Adding Credentials

### For Individual Assessors

1. Navigate to the assessor's profile
2. Click **Add Credential**
3. Select credential type
4. Enter details:
   - Credential name
   - Issuing body
   - Issue date
   - Expiration date
   - Certificate number
5. Upload verification (optional)
6. Save

![Screenshot: Add credential](/img/screenshots/c3pao/add-credential.png)

### Bulk Import

For multiple credentials:
1. Download the import template
2. Fill in credential details
3. Upload the completed file
4. Review and confirm import

## Credential Verification

### Verification Status

| Status | Meaning |
|--------|---------|
| **Verified** | Credential confirmed by issuing body |
| **Pending** | Verification in progress |
| **Unverified** | Not yet verified |
| **Expired** | Credential has lapsed |

### Verification Process

1. Credential information is submitted
2. Platform verifies with issuing authority
3. Status is updated based on verification result

## Expiration Tracking

### Notifications

Receive alerts for expiring credentials:
- 90 days before expiration
- 60 days before expiration
- 30 days before expiration
- On expiration date

### Renewal Process

1. Complete renewal requirements with issuing body
2. Update credential in system
3. Upload new certificate
4. Request re-verification

## Credential Requirements

### For Assessment Roles

| Role | Required Credentials |
|------|---------------------|
| Lead Assessor | CCA + Lead Assessor designation |
| Assessor | CCA |
| Quality Reviewer | CCA or equivalent |

### Per Assessment

Track credential requirements per engagement:
- Lead assessor credential verified
- Team credentials verified
- Minimum team composition met

## Continuing Education

Track CPE/CEU requirements:

- Record continuing education activities
- Track credits earned
- Monitor requirement completion
- Generate CPE reports

## Audit Trail

Maintain credential history:
- When credentials were added
- Verification dates
- Status changes
- Supporting documentation

## Reports

Generate credential reports:

1. Navigate to **Reports > Credentials**
2. Select report type:
   - Current credentials summary
   - Expiration forecast
   - Verification status
   - CPE tracking
3. Download or share

## Best Practices

- **Proactive renewal** - Begin renewal process early
- **Complete records** - Include all relevant credentials
- **Regular review** - Check team credentials monthly
- **Documentation** - Keep certificates and verification on file
