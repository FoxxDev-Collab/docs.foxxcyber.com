# Welcome to Bedrock CMMC C3PAO

Welcome to the Bedrock CMMC C3PAO Assessor Guide. This documentation provides comprehensive guidance for C3PAO assessors using the Bedrock CMMC platform to conduct CMMC assessments.

## What is Bedrock CMMC C3PAO?

Bedrock CMMC C3PAO is the assessor portal of the Bedrock CMMC platform, designed specifically for CMMC Third-Party Assessment Organizations (C3PAOs). It streamlines the assessment process by providing tools to:

- **Manage Assessment Engagements** - Accept and track assessment requests from organizations
- **Review Documentation** - Access and evaluate SSPs, policies, and procedures
- **Assess Controls** - Systematically review CMMC practice implementations
- **Evaluate Evidence** - Review uploaded artifacts and supporting documentation
- **Record Findings** - Document assessment results and recommendations
- **Collaborate with Teams** - Coordinate with lead assessors and team members

## Getting Started

### First-Time Login

When you first access the C3PAO portal, you'll need to:

1. Log in with your assessor credentials
2. Complete your organization profile
3. Set up your marketplace presence
4. Configure team member access

### Dashboard Overview

The C3PAO dashboard provides quick access to:

- Active assessment engagements
- Pending assessment requests
- Recent activity and notifications
- Quick navigation to key features

## Key Features

### Assessment Management

Track all your assessment engagements from initial request through final certification recommendation.

### Control Review

Systematically evaluate each CMMC practice with access to:
- Customer-provided evidence
- Implementation statements
- Supporting documentation

### Team Collaboration

Assign assessors to specific controls or domains, track progress, and consolidate findings.

### Secure Document Access

Review sensitive customer documentation in a secure, controlled environment.
