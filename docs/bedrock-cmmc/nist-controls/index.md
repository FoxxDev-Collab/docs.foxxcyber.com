# NIST SP 800-171 Rev 3 Control Families

NIST Special Publication 800-171 Revision 3 contains **110 security requirements** organized into **14 control families**. These requirements form the foundation for CMMC Level 2 compliance.

## Control Family Summary

| Family | Code | Requirements | Description |
|--------|:----:|:------------:|-------------|
| [Access Control](./access-control) | AC | 22 | Limit system access to authorized users and devices |
| [Awareness and Training](./awareness-training) | AT | 3 | Ensure personnel understand security risks |
| [Audit and Accountability](./audit-accountability) | AU | 9 | Create and protect audit records |
| [Configuration Management](./configuration-management) | CM | 11 | Establish and maintain system baselines |
| [Identification and Authentication](./identification-authentication) | IA | 11 | Verify user and device identities |
| [Incident Response](./incident-response) | IR | 6 | Establish incident handling capabilities |
| [Maintenance](./maintenance) | MA | 6 | Perform and control system maintenance |
| [Media Protection](./media-protection) | MP | 7 | Protect digital and non-digital media |
| [Personnel Security](./personnel-security) | PS | 2 | Screen individuals before access |
| [Physical Protection](./physical-protection) | PE | 6 | Limit physical access to systems |
| [Risk Assessment](./risk-assessment) | RA | 3 | Assess risk to operations and assets |
| [System and Services Acquisition](./system-services-acquisition) | SA | 5 | Allocate resources to protect systems |
| [System and Communications Protection](./system-communications-protection) | SC | 20 | Protect communications at boundaries |
| [System and Information Integrity](./system-information-integrity) | SI | 17 | Identify and correct system flaws |
| **Total** | | **110** | |
