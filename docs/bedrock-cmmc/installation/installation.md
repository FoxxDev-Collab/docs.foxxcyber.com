# Installation Guide

This guide walks you through the installation process for Bedrock CMMC.

## Prerequisites

Before installing Bedrock CMMC, ensure you have:

- An active Bedrock CMMC subscription
- Administrative access to your Microsoft 365 tenant
- A supported web browser (Chrome, Edge, Firefox, or Safari)

## Installation Steps

### Step 1: Access the Bedrock Portal

1. Navigate to the Bedrock CMMC portal
2. Sign in with your organizational credentials
3. Complete the initial authorization flow

### Step 2: Grant Required Permissions

Bedrock CMMC requires certain permissions to assess your environment:

- **Microsoft Graph API** - For reading tenant configuration
- **Azure AD** - For user and group information
- **Exchange Online** - For mail flow and security settings
- **SharePoint Online** - For document and site security

### Step 3: Configure Initial Settings

After granting permissions:

1. Select your target CMMC level (Level 1, 2, or 3)
2. Define your assessment scope
3. Configure notification preferences
