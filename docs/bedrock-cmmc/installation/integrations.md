# Integrations

Bedrock CMMC integrates with various Microsoft 365 and third-party services to provide comprehensive compliance assessments.

## Microsoft 365 Integrations

### Microsoft Entra ID (Azure AD)

Bedrock CMMC connects to Microsoft Entra ID to assess:

- User authentication policies
- Conditional access configurations
- Identity protection settings
- Privileged identity management

### Microsoft Defender

Integration with Microsoft Defender provides:

- Endpoint security assessment
- Threat protection status
- Security recommendations alignment

### Microsoft Purview

Purview integration enables:

- Data classification assessment
- Information protection policy review
- Data loss prevention configuration checks

## Third-Party Integrations

### SIEM Integration

Export assessment data to your SIEM solution:

- Splunk
- Microsoft Sentinel
- Other syslog-compatible systems

### Ticketing Systems

Create remediation tickets automatically:

- ServiceNow
- Jira
- Azure DevOps

## Configuring Integrations

To configure an integration:

1. Navigate to **Settings** > **Integrations**
2. Select the integration you want to configure
3. Follow the connection wizard
4. Test the connection
5. Save your configuration

## API Access

For custom integrations, Bedrock CMMC provides a REST API. See our API documentation for details on:

- Authentication
- Available endpoints
- Rate limits
- Webhook configuration
