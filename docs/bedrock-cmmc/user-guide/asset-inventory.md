# Asset Inventory

Track all systems, devices, and components within your CMMC boundary.

## Managing Assets

Navigate to your package and click **Manage Assets** or access the **Assets** tab.

![Screenshot: Asset inventory](/img/screenshots/cmmc/asset-inventory.png)

## Adding Assets

Click **Add Asset** to create a new inventory item.

### Asset Properties

| Field | Description |
|-------|-------------|
| **Asset Name** | Identifier for the asset |
| **Asset Type** | Hardware, Software, Service, etc. |
| **Description** | Purpose and function |
| **Location** | Physical or logical location |
| **Owner** | Responsible party |
| **Risk Rating** | Criticality assessment |

![Screenshot: Add asset form](/img/screenshots/cmmc/add-asset.png)

## Asset Types

Common asset categories to track:

- **Hardware** - Servers, workstations, network devices
- **Software** - Applications, operating systems
- **Services** - Cloud services, managed services
- **Data** - Databases, file stores, CUI repositories
- **Personnel** - Users with system access
