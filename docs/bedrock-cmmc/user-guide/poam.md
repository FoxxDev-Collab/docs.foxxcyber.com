# Plan of Action & Milestones (POA&M)

POA&Ms track remediation efforts for identified compliance gaps.

## Viewing POA&Ms

Navigate to **POA&Ms** from the main menu and select your package.

![Screenshot: POA&M list view](/img/screenshots/cmmc/poam-list.png)

### POA&M Statistics

The dashboard shows:

- **Total POA&Ms** - All remediation items
- **Open** - Items requiring attention
- **In Progress** - Currently being remediated
- **Completed** - Successfully closed items

## Creating a POA&M

Click **New POA&M** to create a remediation plan.

![Screenshot: Create POA&M form](/img/screenshots/cmmc/create-poam.png)

### Required Fields

| Field | Description |
|-------|-------------|
| **Title** | Brief description of the weakness |
| **Description** | Detailed explanation of the gap |
| **Associated Requirements** | Link to relevant NIST requirements |
| **Status** | Open, In Progress, Completed, Closed |
| **Risk Level** | Critical, High, Moderate, Low |
| **Target Completion Date** | When remediation should be complete |

## Adding Milestones

Break down remediation into trackable milestones:

1. Click **Add Milestone** within the POA&M
2. Enter milestone description
3. Set milestone due date
4. Track completion status

![Screenshot: POA&M milestones](/img/screenshots/cmmc/poam-milestones.png)

## POA&M Status Workflow

```
Open → In Progress → Completed → Closed
         ↓
      Overdue (if past deadline)
```
