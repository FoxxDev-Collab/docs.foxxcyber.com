# C3PAO Marketplace

Find and engage certified assessors for your CMMC assessment.

## Browsing the Marketplace

Navigate to **Marketplace** from the main menu.

![Screenshot: C3PAO Marketplace](/img/screenshots/cmmc/marketplace.png)

## Filtering C3PAOs

Use filters to find the right assessor:

- **Search** - By name, location, or specialty
- **CMMC Level** - Filter by authorized assessment levels
- **Specialties** - Industry-specific experience

## C3PAO Profiles

Each listing displays:

| Information | Description |
|-------------|-------------|
| **Organization Name** | C3PAO company name |
| **Location** | City and state |
| **Description** | Services and approach |
| **Authorized Levels** | CMMC levels they can assess |
| **Rating** | Customer reviews and score |
| **Typical Timeline** | Expected assessment duration |
| **Cyber-AB ID** | Accreditation identifier |

![Screenshot: C3PAO profile detail](/img/screenshots/cmmc/c3pao-detail.png)

## Requesting an Assessment

### Step 1: Select C3PAO

From the marketplace, click **View Details** on your chosen C3PAO, then click **Request Assessment**.

### Step 2: Select Package

Choose which ATO package you want assessed.

![Screenshot: Request assessment form](/img/screenshots/cmmc/request-assessment.png)

### Step 3: Add Notes (Optional)

Include any relevant information for the assessor:

- Preferred timeline
- Special considerations
- Contact preferences

### Step 4: Submit Request

Click **Submit Request** to send to the C3PAO.
