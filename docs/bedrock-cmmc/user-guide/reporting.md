# Reporting

Bedrock CMMC provides comprehensive reporting capabilities to communicate compliance status to stakeholders, auditors, and leadership.

## Report Types

### Executive Summary

A high-level overview designed for leadership:

- Overall compliance score
- Trend over time
- Key risk areas
- Recommended priorities

### Detailed Compliance Report

A comprehensive report for compliance teams:

- Control-by-control status
- Evidence documentation
- Gap analysis
- Remediation status

### Auditor Package

Documentation package for CMMC assessors:

- System Security Plan (SSP) data
- Plan of Action & Milestones (POA&M)
- Evidence artifacts
- Assessment history

### Custom Reports

Build custom reports based on:

- Specific control families
- Date ranges
- Organizational units
- Compliance trends

## Generating Reports

To generate a report:

1. Navigate to **Reports** in the main menu
2. Select the report type
3. Configure parameters:
   - Date range
   - Scope
   - Format (PDF, Excel, Word)
4. Click **Generate**

## Scheduling Reports

Automate report delivery:

1. Go to **Reports** > **Scheduled**
2. Click **New Schedule**
3. Select report type and parameters
4. Set frequency (daily, weekly, monthly)
5. Add recipients
6. Save the schedule

## Report Distribution

Reports can be distributed via:

- Email (with optional encryption)
- SharePoint upload
- Secure download link

## Data Retention

Reports are retained according to your organization's settings:

- Default: 2 years
- Configurable in **Settings** > **Data Retention**

## Compliance Evidence

Each report includes traceable evidence:

- Timestamp of data collection
- Source systems queried
- Configuration snapshots
- User attestations
