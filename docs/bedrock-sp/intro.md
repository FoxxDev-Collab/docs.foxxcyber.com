# Welcome to Bedrock SP

Bedrock SP (Security Platform) is a comprehensive, enterprise-grade security operations platform that unifies threat detection, incident response, and security orchestration in a single integrated solution.

## What is Bedrock SP?

Bedrock SP provides:

- **Unified Security Operations** - Centralized security operations center (SOC) platform
- **SIEM Capabilities** - Advanced security information and event management
- **SOAR Integration** - Security orchestration, automation, and response
- **Threat Intelligence** - Real-time threat intelligence feeds and analysis
- **Incident Management** - End-to-end incident response workflow
- **Forensics & Investigation** - Digital forensics and investigation tools

## Key Features

### Advanced Threat Detection
Machine learning-powered threat detection with behavioral analytics and anomaly detection.

### Automated Response
Pre-built playbooks for automated incident response and threat containment.

### Threat Hunting
Proactive threat hunting capabilities with advanced query language and visualizations.

### Integration Ecosystem
Integrate with 300+ security tools and platforms:
- EDR/XDR solutions
- Firewalls and network devices
- Cloud platforms
- Identity providers
- Ticketing systems
