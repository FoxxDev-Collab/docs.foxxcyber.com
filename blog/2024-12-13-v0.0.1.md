We're excited to announce the initial release of Bedrock CMMC, our comprehensive compliance management platform designed to help organizations achieve and maintain CMMC certification.

<!-- truncate -->

## What's Included

This foundational release includes:

### ATO Package Management
- Create and manage Authorization to Operate packages
- Track package status and completeness
- Organize compliance documentation

### Control Assessment
- Full NIST SP 800-171 Rev 3 control library (110 requirements)
- Assess implementation status for each control
- Track compliance progress by control family

### Evidence Management
- Upload and organize compliance evidence
- Link evidence to specific controls
- Maintain audit-ready documentation

### POA&M Tracking
- Create and manage Plans of Action & Milestones
- Set remediation timelines and milestones
- Track progress on identified gaps

### SSP Generation
- Generate System Security Plans
- Track SSP readiness
- Export documentation for assessors

### Asset Inventory
- Catalog system components
- Track assets within scope
- Maintain hardware and software inventory

### C3PAO Marketplace
- Browse certified assessor organizations
- Request assessments
- Manage assessor engagements

## What's Next

We're actively developing new features and improvements. Stay tuned for updates as we prepare for our public beta launch in January 2026.
